AR Safety Signs - mobile app for safety training using augmented reality
Watch a demo video on YouTube

AR Safety Signs

Technologies:

  • C#
  • Unity 2019.3
  • Unity AR Foundation 3.0
    • Image tracking
    • Plane detection
    • Raycasting
    • Anchors

A prototype of a fire safety mobile app using augmented reality. It consists of five mini games that are activated by scanning a safety sign. I created the app during a two-month student training period in Turku Game Lab with two other team members. I did most of the scripting (C#) and three mini games were completely my creations: the alarm button game, the fire extinguishing game and the bonus feature.

The prototype is now property of Turku Game Lab but you can watch my YouTube video that demonstrates its features.

Värimetsä - endless runner style color learning game in Finnish
Watch a game play video on YouTube

Värimetsä

Technologies:

  • C#
  • Unity 2019.3

Värimetsä is an endless runner style color learning game for 3-5-year-olds. I had a game development course in H2C (Highway2Code) studies and this was my project work. The game is still missing many features, most importantly Finnish audio so that the young ones would understand the tasks. I'm working on the touch controls and mobile optimization but at the moment it is only playable on desktop with stand-alone player or WebGL.

You can watch my Youtube video to see the basic game play.

There was also an assignment to create a responsive website for the game. You can visit the Värimetsä website now or see the details later in this portfolio.

Komero

Technologies:

  • Progressive Web App
  • React with JavaScript
  • Google Firebase
  • (SQL Server)
  • (ASP.Net Core)

Komero web app helps with managing children's seasonal outerwear and equipment. I use this app during offseason to see what items and sizes we already have in our storage boxes.

My original version from 2019 uses Google Firebase. You can try it or look at the code on GitHub. I also made another version of this app during Saranen Code Bootcamp (April 2020) using the back-end technologies that we studied (SQL Server and web api with ASP.Net Core). It's still missing authentication and lacks some testing.
